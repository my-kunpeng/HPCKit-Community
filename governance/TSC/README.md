# 技术指导委员会（TSC-Technical Steering Committee）定位与职责
- 定义HPCKit项目的技术发展方向和愿景，决策重大技术事项；
- 制定、修改技术指导委员会相关制度；
- 选举、任免技术指导委员会主席及委员;
- 决策SIG（Special Interest Group 特别兴趣小组）的成立和撤销；
- 其他对社区有重要影响的技术工作。

## 会议准则  
会议频次：月度，欢迎订阅[TSC邮箱]()了解相关会议公告及内容，我们欢迎任何感兴趣的开发者旁听会议。

## 会议纪要  
您可以在[TSC会议纪要]()查看到相关内容，TSC并会在每次会议后将会议纪要发布到邮件列表中。
